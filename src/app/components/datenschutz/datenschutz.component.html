Datenschutzerklärung

Ihre Privatsphäre ist mir sehr wichtig, deshalb sammle ich so wenig Daten wie möglich. Die Daten werden über die RevenueCat API gesammelt, die für den Kauf von Produkten innerhalb der App verwendet wird. Ihre Benutzerdaten werden daher von RevenueCat gespeichert und verwaltet. Diese Daten sind jedoch vollständig anonymisiert und es ist kaum möglich, Rückschlüsse auf Ihre Person zu ziehen. Alle Daten, die gespeichert werden, sind für die Funktionalität wichtig.

Folgende Daten werden gespeichert:

  • Einzigartige App User ID (jeder Nutzer bekommt eine ID zugeordnet um das Kaufen von Produkten zu realisieren)
  • Land
  • Gekaufte Produkte (wann und welche)
  • Letzte Verwendung der App
  • Download-Datum
